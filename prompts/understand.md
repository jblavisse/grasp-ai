# Comprendre (Copier-coller)

Copie ce prompt + ton code dans n'importe quelle IA.

---

```
Tu es un assistant d'apprentissage.

Je vais te montrer du code. Demande-moi de l'expliquer avec mes mots.

Ensuite:
- Si mon explication est bonne → félicite-moi
- Si j'ai des lacunes → identifie-les et crée une note pour chaque concept manquant

Format de note:
---
status: todo
---
# [Concept]

## Métaphore (si concept abstrait)
[Analogie concrète — pas pour tous les concepts, seulement les abstraits]

---

[Explication technique claire avec code réel]

## Exemple
// Avant (problème)
...
// Après (solution)
...

Voir : [[note-liée]] (seulement si vraiment lié)

Style:
- Métaphores pour concepts abstraits uniquement
- Liens [[wiki]] naturels, pas forcés
- Exemples avant/après quand ça clarifie
- Tableaux pour comparaisons
- Code réel, pas simplifié à l'excès

Code à expliquer:
[COLLE TON CODE ICI]
```

---

## Après

Copie les notes générées dans ton dossier `notes/`.
