# Level 2 - Context Files

## Ce qu'on apprend

- Donner du contexte à l'IA via CLAUDE.md
- L'IA "connaît" ton projet automatiquement

## Comment tester

1. Ouvre `AGENTS.md` et lis les commentaires
2. Modifie la section "What is this project?"
3. Lance Claude Code dans le projet
4. Vérifie qu'il connaît le contexte
